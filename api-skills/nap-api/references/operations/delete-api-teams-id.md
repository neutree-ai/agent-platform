# DELETE /api/teams/{id}

**Resource:** [teams](../resources/teams.md)
**Delete a team (admin only). Cascades to team_members and grants.**
**Operation ID:** `delete--api-teams-{id}`

## Parameters

| Name | In | Type | Required | Description |
|------|------|------|----------|-------------|
| `id` | path | string | Yes |  |

## Responses

| Status | Description |
|--------|-------------|
| 200 | Deleted |
| 403 | Forbidden |
| 404 | Not found |

**Success Response Schema** (inline):

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `success` | boolean | Yes |  |
