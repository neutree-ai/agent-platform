## Appearance

自定义 NAP 的外观和提示音。

### Theme

- **Light / Dark** — 固定配色
- **System** — 跟随操作系统设置，自动切换明暗模式

### Sound

开启或关闭提示音。当 agent 完成一轮执行、将控制权交还给您时会播放提示音。

### Language

切换界面语言。仅影响 UI 文本，不影响 agent 生成的内容。

### Chat send key

切换聊天输入框中 Enter 和 Cmd/Ctrl+Enter 的行为。

- **⌘/Ctrl + Enter**（默认）：Cmd/Ctrl+Enter 发送，Enter 换行
- **Enter**：Enter 发送，Shift+Enter 换行（与 Slack 等工具一致）

设置存储在用户级别，跨设备同步。仅作用于工作区聊天输入框；Prompt 编辑、Agent 配置等多行文本编辑器不受影响。
