快捷创建 SSH 私钥凭据。选择密钥类型后会自动填充标准的文件名和路径，以 `0600` 权限写入容器。

## 支持的密钥类型

| 类型 | 文件名 | 路径 |
|------|--------|------|
| ED25519 | `id_ed25519` | `~/.ssh/id_ed25519` |
| RSA | `id_rsa` | `~/.ssh/id_rsa` |
| ECDSA | `id_ecdsa` | `~/.ssh/id_ecdsa` |
| Custom | 自定义 | 自定义 |

## 用法

将私钥内容粘贴到 **Value** 字段中。通常从本地 `cat ~/.ssh/id_ed25519` 获取。

## 注意事项

- 仅需粘贴**私钥**，公钥需要在 Git 平台侧配置
- 平台会自动补末尾换行符
- 如需同时配置多个密钥，创建多条 credential 即可
