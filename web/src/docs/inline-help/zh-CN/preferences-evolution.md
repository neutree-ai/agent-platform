## Auto Evolution

开启后，NAP 会分析已完成的 agent 会话，自动建议 prompt、工具和执行流程的优化。

### 工作原理

1. 每次会话结束后，NAP 会回顾对话内容
2. 如果发现可优化的地方，会生成优化建议
3. 您可以在 workspace 面板中查看并决定是否采纳

### 适合开启的场景

- 您正在迭代 agent 行为，希望基于数据驱动改进
- Workspace 运行重复性任务，小幅优化可以积累出明显效果

### 适合关闭的场景

- Workspace 的 prompt 已经定稿，不希望变动
- 您希望完全手动控制所有 prompt 变更
