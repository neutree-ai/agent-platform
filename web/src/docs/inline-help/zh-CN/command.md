自定义命令可以将常用指令封装为快捷操作。在聊天输入框中输入 `/` 即可触发。

## 命令类型

- **Plain** — 固定文本，触发后直接发送给 agent
- **Struct** — 模板，包含 `{{变量}}` 占位符，触发时弹出表单让用户填写

## Struct 模板语法

使用双花括号定义变量：

```
项目 {{PROJECT_ID}} 的 {{BRANCH}} 分支，请检查 {{FILE_PATH}} 文件
```

触发时会弹出表单，包含 `PROJECT_ID`、`BRANCH`、`FILE_PATH` 三个输入框。

变量名建议使用大写 + 下划线命名。

## 内容来源

- **Custom** — 直接在此编辑内容
- **Prompt Library** — 链接到 prompt 库中的条目，内容随 prompt 更新自动同步

## 注意事项

- 命令名不能与内置命令重名
- 链接 Prompt 后，内容由 prompt 库管理，此处只读
