周期定时任务按 cron 表达式重复触发，每次触发时在 workspace 中创建一个新 session 并执行 **Prompt** 中的指令。

## Cron 表达式

标准五段格式：`分 时 日 月 周`

| 示例 | 含义 |
|------|------|
| `0 9 * * *` | 每天 9:00 |
| `0 9 * * 1-5` | 工作日 9:00 |
| `*/30 * * * *` | 每 30 分钟 |
| `0 0 1 * *` | 每月 1 号 0:00 |

## 字段

- **Name** — 任务名称，用于列表展示和日志标识
- **Cron** — 触发节奏
- **Timezone** — cron 表达式的时区基准，默认为浏览器时区
- **Prompt** — 每次触发时发送给 agent 的指令内容；可直接填写文本，也可引用 prompt 库中的模板

## 注意事项

- 禁用后不再触发，但保留配置
- 每次触发都是独立 session，不共享上下文
- 最小间隔建议不低于 5 分钟
- 在 agent 对话里也可以让 agent 帮你创建、修改、删除周期任务
