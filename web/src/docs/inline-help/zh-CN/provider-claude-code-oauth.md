使用你个人的 Claude 订阅（Pro / Team）—— 不需要服务商 API Key，也不需要 Base URL。供 Claude Code agent 使用。

在自己的机器上（已安装并登录 Claude Code CLI）执行：

```bash
claude setup-token
```

把输出的 token 粘贴到 **API Key** 输入框。

## 适用场景

- 拥有 Claude Pro / Team 订阅，但没有团队 API Key

## Visibility

- **Private**: 仅自己可见
- **Team**: 共享给所选团队
- **Public**: 平台所有用户可见、可用
