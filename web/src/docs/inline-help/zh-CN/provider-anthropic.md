直连 Anthropic 官方 API，使用静态 API Key 认证。适合团队统一采购 API 额度后共享给成员使用。

- **Base URL**: `https://api.anthropic.com`
- **API Key**: 从 [console.anthropic.com](https://console.anthropic.com) 获取

## 适用场景

- 团队有 Anthropic API 额度，需要集中管理用量
- 支持 Claude Code agent

## Visibility

- **Public**: 平台所有用户可见、可用
- **Private**: 仅自己可见
