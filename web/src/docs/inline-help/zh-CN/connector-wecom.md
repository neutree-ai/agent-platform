连接企业微信，通过智能机器人 WebSocket 长连接实时接收群聊 @机器人 消息。

## 所需凭据

- **Bot ID** (`aibXXX...`) — 智能机器人的唯一标识
- **Secret** — 智能机器人的长连接 Secret（注意：不是自建应用的 corpsecret）

## 创建智能机器人

1. 登录企业微信管理后台
2. 前往 **安全与管理 → 管理工具 → 智能机器人**
3. 创建机器人，选择 **API 模式**
4. 记录 Bot ID 和 Secret
5. 将机器人添加到目标群聊（群需至少 3 人）

## 下一步

Connector 创建后，需要为具体的群聊创建 **Route** 来定义：
- 监听哪个群（群聊 ID）
- 触发哪个 Workspace 执行任务
- 如何将消息转为 prompt（模板）

## 注意事项

- 被动回复窗口为 24 小时，且每条消息的 req_id 仅可回复一次
- 频率限制：30 条/分，1000 条/时
- 企业微信无原生 thread，平台通过时间窗口自动管理会话
