## About

This page shows version information for the current web build.

- **Current version**: The commit corresponding to the bundle loaded in the browser
- **Build time**: The time this image was packaged in CI (UTC)
- **Server version**: The latest version currently served by the server

If the current version differs from the server version, a new version has been deployed. The system automatically detects this and prompts you to refresh, or you can click "Refresh" manually to load the latest code.

Upgrade / self-deployment verification: by comparing the versions here, you can quickly confirm whether the service connected to this machine has switched to the new image.
