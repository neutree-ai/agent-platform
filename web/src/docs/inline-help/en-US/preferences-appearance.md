## Appearance

Customize NAP appearance and notification sound.

### Theme

- **Light / Dark** — Fixed color scheme
- **System** — Follow the operating system setting and switch between light and dark modes automatically

### Sound

Enable or disable notification sound. A sound plays when the agent completes one round of execution and returns control to you.

### Language

Switch the interface language. This only affects UI text and does not affect content generated by the agent.

### Chat send key

Switch the behavior of Enter and Cmd/Ctrl+Enter in the chat input.

- **⌘/Ctrl + Enter** (default): Cmd/Ctrl+Enter sends, Enter inserts a newline
- **Enter**: Enter sends, Shift+Enter inserts a newline (consistent with tools such as Slack)

The setting is stored at the user level and synchronized across devices. It only applies to the Workspace chat input; multiline text editors such as prompt editing and Agent configuration are not affected.
