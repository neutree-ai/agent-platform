Tags are used to group Workspaces and filter them quickly.

## Usage

1. **Create a tag** — Click New Tag on this page and set its name and color
2. **Apply tags** — Select tags in the Workspace menu (three-dot button) on the Dashboard
3. **Filter by tag** — The tag buttons at the top of the sidebar can quickly filter the Workspace list

## Design suggestions

- Classify by **purpose**: `production`, `staging`, `experiment`
- Classify by **team**: `frontend`, `backend`, `data`
- Classify by **status**: `active`, `archived`, `review`

Colors can be used to distinguish different categories visually.

## Notes

- Deleting a tag automatically removes it from all Workspaces
- Tag filtering uses OR logic — when multiple tags are selected, matching any one displays the Workspace
